MVC Micro Framework

Third Parties Components

MVC Micro Framework allows the use of components of third-parties. It implements some components in the base.

HttpFoundation

Symfony

Thanks to HttpFoundation it is possible to access http communication with OOP approach. Detailed documentation of using HttpFoundation can be found here.

Eloquent ORM

Laravel

Eloquent allows us to use ORM mapping. More information about using Eloquent can be found here.

Monolog

Logger

Monolog is currently the most popular tool for logging in PHP system. More information here.

DI Container

Pimple

As an implementation of the Dependency Injection design pattern, we will launch the most widely used DI container on the market. Pimple. Detailed documentation here.
