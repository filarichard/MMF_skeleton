Event Dispatcher

Event Dispatcher je další důležitou součástí v MVC Micro Frameworku. Jedná se o jednoduchou implementaci návrhového vzoru Observer.

Event Dispatcher

Event

Event Dispatcher je jednoduchou implementací návrhového vzoru Observer a vychází ze standardů PSR-3. Pro jeho užití je jednoduché. Stačí vytvořit objekty typu Event a Dispatcher.

Dispatch

U Dispatcheru se poté zaregistruje Listener, což je jakákoliv spustitelná část kódu. Dále stačí kdekoliv v kódu zavolat metodu objektu Dispatcher, Dispatch, která všechny zainteresované listenery informuje o skutečnosti, která nastala. Dále je možné jakoukékoliv události zastavit propagaci. Tedy pakliže nastane daná událost, posluchačům se nedá vědět.
